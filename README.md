# Запуск проекта с помощью docker-compose:
1. Клонируйте проект
```shell
gi clone https://github.com/Ilya96s/job4j_dish
```
2. Перейдите в папку проекта
```shell
cd job4j_dish
```
3. Создайте образ проекта
```shell
docker-compose build
```
4. Выполните команду
```shell
docker-compose up
```
